Quick Start

First, install the app.

  • Android
  • iOS

For Application Integrators:

1. Log into the application and create an entry on the Applications screen with your application's name, allowed hosts and OAuth CLIENT_ID.

2. If your application uses a service with an OpenAI compatible interface, just update your code to use api.keytrustee.org in place of the current API URL. If your application is using a SDK or non-OpenAI compatible endpoint, you'll need to convert it to a http request using the OpenAI API interface with that URL.

3. Whenever you need to use an API, just pass the user's OAuth token in the authorization header instead of your API key. Optionally, you can update the model parameter to an ordered preference array so that a good alternate model can be selected when a user doesn't have access to your preferred provider - otherwise you will get whatever default model the user has selected.

For Users:

1. Log into the app with your primary identity provider. If you use multiple identity providers, visit the identites page and add them to your account to ensure your keys are available regardless of which provider you log into an application with.

2. Add your credit card information. Key Trustee doesn't charge any fees, however almost all API providers require payment information, and we will supply the card you provide to providers automatically on your behalf.

3. Go to the provider configuration screen and click the button for any providers you would like automatically configured for you and made available to applications.
